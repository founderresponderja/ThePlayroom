# App Icons

Replace these files before submitting to app stores:
- icon.png (1024x1024) — main app icon
- adaptive-icon.png (1024x1024) — Android adaptive icon foreground
- splash.png (1284x2778) — splash screen

Current placeholder uses the default Expo icon.
For production, use a pineapple design with:
- Background: #0B0708
- Pineapple body: #FF1F3D
- Leaves: #4ade80
- Accents: #FF2E9A
